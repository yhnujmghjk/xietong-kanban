# 协同看板

基于设计稿实现的协同看板单页应用，包含三个主要页面：
- 协同看板（项目进度矩阵）
- 人员库（团队成员管理）
- 数据面板（导入/导出）

## 本地运行

```bash
python -m http.server 8080
# 访问 http://localhost:8080
```

## 数据同步说明

本项目为纯前端应用，数据仅存储在浏览器内存中。多人协作流程：
1. 用户A编辑完成后，点击数据面板"导出报表"下载JSON文件
2. 将JSON文件发送给用户B
3. 用户B点击"导入数据"加载文件，继续编辑

## 文件结构

- `index.html` - 主页面，整合三个子页面
- `styles.css` - 全局样式
- `app.js` - 交互逻辑
